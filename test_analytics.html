Google Analytics Event Tracking Test

Click the buttons below to test the event tracking. Check your browser's developer console (F12) to see the events being sent to Google Analytics.

Instructions:

  1. Open your browser's Developer Tools (F12)
  2. Go to the Console tab
  3. Click each button above
  4. You should see console messages confirming the events were sent
  5. Check the Network tab to see the requests being sent to Google Analytics

Event Details:

  • hub_click - Tracks clicks on the OOTB Analytics Hub project (both card and View Project button)
  • chart_click - Tracks clicks on the Chart Builder Transformation project (both card and View Project button)
  • delivery_click - Tracks clicks on the Delivery Operations Tools project (both card and View Project button)
  • transaction_click - Tracks clicks on the Transaction Platform project (both card and View Project button)

Event Labels:

  • All clicks (card + button): ootb_analytics_hub, chart_builder_transformation, delivery_operations_tools, transaction_platform
  • Note: Both card clicks and "View Project" button clicks use the same event labels for consistent tracking

Note: All events are categorized under 'project_engagement' and have a value of 1 for easy tracking in Google Analytics.
